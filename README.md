# Snake-OOP-DualPlayer 🐍
> 一个基于 Python & Pygame 开发的、面向对象架构的、支持双人竞技与动态难度系统的增强版贪吃蛇。
> A refactored Snake game featuring OOP design, dual-player mode, and dynamic difficulty scaling.

---

## 🌟 项目亮点 (Features)

- **[OOP] 面向对象设计**：代码采用模块化封装，解耦了游戏逻辑、UI 渲染与资源管理，易于扩展。
- **[Dual-Player] 同屏双人**：支持 P1 (WASD) 与 P2 (方向键) 同屏博弈，具备完善的交叉碰撞检测。
- **[Dynamic Difficulty] 动态难度**：内置非线性难度梯度算法，随得分增长自动调整全局帧率 (FPS)。
- **[Skin System] 皮肤切换**：数据驱动的皮肤管理系统，支持通过配置列表快速增加新角色。
- **[macOS Optimized] 系统适配**：针对 macOS 系统进行了 Retina 屏幕渲染优化及字体兼容处理。

---

## 🛠️ 技术栈 (Tech Stack)

- **Language**: Python 3.10+
- **Library**: Pygame (Game Engine)
- **Design Pattern**: Object-Oriented Programming (OOP)
- **Data Handling**: JSON/Txt for Highscore Persistence
